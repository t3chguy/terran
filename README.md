# Terran

> “You wanna piece of me, boy?” — _Marine_

[Powerline] prompt based on the Terran race.

![Terran]

## Install

With [fisherman]

```fish
fisher terran
```

## Terminal Settings

* [Terran.terminal]
* [Terran.itermcolors]
* Font: 16pt Source Code Pro Medium
* Non-ASCII Font: 17pt Source Code Pro for [Powerline]
