# Terran

> “You wanna piece of me, boy?” — _Marine_

A prompt based on the Terran race from StarCraft.

![Terran]

## Install

With [Fisherman]

```fish
fisher i terran
```

## Terminal Settings

* [Terran.terminal]
* [Terran.itermcolors]
* Font: 16pt Source Code Pro Medium
* Non-ASCII Font: 17pt Source Code Pro for [Powerline]
